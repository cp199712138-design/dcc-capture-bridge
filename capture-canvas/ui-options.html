DCC Capture Bridge - UI 方向选择

先选产品气质，再继续做功能。当前只做方向稿，不锁死实现。

A

Capture Studio

最接近专业 DCC 工具。深色、克制、信息密度适中，适合先做产品级 MVP。

  • 适合 3ds Max / Blender 用户，不像玩具。
  • 左侧采集，中间证据板，右侧 AI 任务，底部历史。
  • 推荐作为第一版主方向。

B

Realtime Canvas

更像 Krea / Vectary 的演示感。中心区域更大，更强调实时生成和参考槽。

  • 适合后续接 ComfyUI 实时刷新。
  • 视觉更酷，但第一版开发量更高。
  • 适合做官网演示和客户展示。

Product Board

更像品牌设计板/产品简报。浅色画布、注释卡、素材整理能力强。

  • 适合客户评审和方案沟通。
  • 不够 DCC 工具感，但更高级、更干净。
  • 适合后续做报告、对比、交付页面。
