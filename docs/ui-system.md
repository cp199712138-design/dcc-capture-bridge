# UI System Direction

## Product Type

```text
Creative professional tool
DCC utility
Rendering workflow helper
Future AI pipeline control
```

## Chosen Style

```text
Precision Studio UI
```

Principles:

- Dark-first, graphite professional interface.
- Compact inspector layout.
- High signal, low decoration.
- One primary action per screen.
- Clear status text, not color-only feedback.
- No glassmorphism, no neon HUD, no marketing card layout.

## UI Levels

### Level 1: Compact Plugin

The MVP panel:

```text
Capture / Render
Viewport detection
Resolution presets
Output format and folder
Save image
Status
```

### Level 2: Capture + AI Panel

Later:

```text
Capture / Render / AI / Review
ComfyUI status
Workflow preset
Prompt
Generate
Output preview
```

### Level 3: Canvas Workspace

Later advanced workspace:

```text
Modes + cameras
Preview canvas
Inspector settings
Recent outputs
Batch queue
Multi-angle variants
```

## Language Strategy

English is the default UI and development language. Chinese support is planned, but MAXScript encoding needs testing before shipping bilingual UI strings directly in the script.
